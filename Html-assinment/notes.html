5 html tags I have learnt and their description

Tag p
Sort for Paragraph The tag is used to define a paragraph of text.
Tag h1, h2, h3, h4, h5
Sort for Headings and these tags are used to create heading lins with diffrent levels of importance.
Tag table
Used to create tables on a web page. It contains rows to organize data in rows and columns.
Button tag
Used to create a clickable button on a web page.
title tag
Sets the title of the web page.
The most important tags to properly make a website and to remember are Tag >!DOCTYPE, Tag html, Tag head, Tag meta and Tag body.
